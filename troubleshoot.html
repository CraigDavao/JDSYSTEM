Jolly Dolly Troubleshooting Guide

Identify and fix issues with your CSS and JavaScript

Common Issues

1. File Path Problems

The most common issue is incorrect file paths. Check if your paths are correct:

  • Absolute path: /jollydolly-website/css/style.css
  • Relative path: ../css/style.css (depends on file location)

2. File Permissions

Ensure your CSS and JS files have proper read permissions:

  • On Linux: chmod 644 css/style.css
  • On Windows: Check file properties

3. Browser Caching

Your browser might be loading old cached versions:

  • Hard refresh with Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  • Clear browser cache entirely

4. Syntax Errors

Check for errors in your CSS/JS files:

  • Use browser developer tools (F12)
  • Check the Console tab for JavaScript errors
